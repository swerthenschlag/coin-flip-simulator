# Coin Flip Simulator

A Python program that simulates coin flips and compares experimental probabilities to theoretical probabilities.

## Features

- User chooses the number of coin flips
- Simulates heads and tails outcomes
- Counts total heads and tails
- Calculates experimental probabilities

## Example Output

Expected Heads = .5000
Experimental Heads = 0.5017

Expected Tails = .5000
Experimental Tails = 0.4983

## Concepts Used

- Variables
- Loops
- Conditionals
- User input
- Random number generation
- Probability simulation
